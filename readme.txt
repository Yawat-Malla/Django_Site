A simple website developed for Key Club of Kathmandu on Django framework. 

Django is a high-level web framework for building web applications using Python.

This project is a simple webapp developed using html,css and javascript. The website is made easily modifiable for any changes/improvements in the future. 
  

The backend is mainly responsible for directing the users to the respective webpages. All the webpages or user-visible contents are stored in the "templates" folder. The static files for the webpage(files except the content page which has to be loaded like css or images) is stored in the "static" folder.
